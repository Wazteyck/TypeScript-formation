## Install

Open a command line tool (cmd).

```
git clone https://github.com/jlg-formation/tsr-juillet-2021.git
cd tsr-juillet-2021
cd back
npm i
cd ..
cd front
npx webpack
cd ..
cd back
npm start
```

## Run

Open a web browser on `http://localhost:3000/`
