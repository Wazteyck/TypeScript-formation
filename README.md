## Install

Open a command line tool (cmd).

```
git clone https://github.com/jlg-formation/tsr-juillet-2021.git
cd tsr-juillet-2021
cd ..
cd front
npm i
npm start
```

Open a second command line tool.

```
cd 'folder'
cd back
npm i
npm start
```

## Run

Open a web browser on `http://localhost:3000/`

## Production

```
git clone https://github.com/jlg-formation/tsr-juillet-2021.git
cd tsr-juillet-2021
cd ..
cd front
npm i
npm run build
cd ..
cd back
npm i
npm run compile
npm i -g pm2
pm2 start ecosystem.config.js
```
